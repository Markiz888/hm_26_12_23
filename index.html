git clone-используется для создания копии (клонирования) удалённого репозитория Git

git init- используется для инициализации нового репозитория Git внутри существующей директории

git config --global user.name-используется для просмотра глобального настройки имени пользователя в Git

git config --global user.email-используется для просмотра глобального настройки электронной почты пользователя в Git

git add- используется для инициализации нового репозитория Git внутри существующей директории

git commit- используется для создания коммита в Git

git push-используется для отправки ваших локальных изменений в удаленный репозиторий
